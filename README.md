# Propflow
Automated email marketing for real estate agents and mortgage brokers.

## About
Propflow is an email marketing tool designed for real estate agents and mortgage brokers.  
It helps improve client acquisition and retention through newsletters and automated emails that trigger as prospects move through the sales pipeline.

## How It Works

### Example: Mortgage Broker Workflow
- Documents Received → Confirmation email sent
- Prequalification Completed → Follow-up email sent
- Mortgage Closed → Review request email sent
- Start of Month → Monthly newsletter sent

All emails are automatically triggered based on pipeline status, ensuring communication without manual follow-ups.
